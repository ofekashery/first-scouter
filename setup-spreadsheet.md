# Setup the Spreadsheet

#### 1. Open your Spreadsheet,
Open your Spreadsheet or [create one](http://spreadsheet.new).

#### 2. Open the Script Editor
Open the **Script editor** by clicking **"Tools" > "Script editor"**.
<br/><img width="250" src="https://user-images.githubusercontent.com/16443111/59484331-f2f76f80-8e79-11e9-8f8a-d6a3ad0caefa.png" />

#### 3. Setup the code
Paste the [this](https://raw.githubusercontent.com/ofekashery/first-scouter/master/google-apps-script.gs) code.
<br/>Then, click **"Run" > "Run function" > "setup"**
<br/><img width="250" src="https://user-images.githubusercontent.com/16443111/59484479-a3fe0a00-8e7a-11e9-8b27-1e565d3eb778.png" />

#### 4. Last step - Deploy
Deploy the script Click by clicking **Publish > Deploy as web app**.
<br/>Set security level and enable service (execute as 'me' and access 'anyone, even anonymously')
<br/>Then, copy the 'Current web app URL' to the environment files (Detailed explanation in the [README.md](https://github.com/ofekashery/first-scouter/blob/master/README.md#setting-up-the-settings) file).
<br/><img width="250" src="https://user-images.githubusercontent.com/16443111/59484854-26d39480-8e7c-11e9-9678-d835caa8c65f.png" />
